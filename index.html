Enlace de correo

El enlace mailto:

La forma más directa de invitar al visitante a contactarte. Un solo atributo convierte cualquier texto en un botón que abre el cliente de correo del usuario.

¿Para qué sirve?

Cuando colocas mailto: en el atributo href de un enlace, el navegador entiende que al hacer clic debe abrir la app de correo del dispositivo (Mail, Outlook, Gmail, etc.) con el destinatario ya rellenado. Puedes además pre-rellenar el asunto y el cuerpo del mensaje.

<!-- Enlace básico --> <a href="mailto:[email protected]"> Escríbeme </a> <!-- Con asunto y cuerpo pre-rellenados --> <a href="mailto:[email protected] ?subject=Hola desde tu web &body=Me interesa tu servicio..."> Contactar ahora </a>

Ventaja: sin backend, sin código extra. Limitación: depende de que el usuario tenga un cliente de correo configurado.

Anatomía web

Estructura, estilo y comportamiento

Cada tecnología web tiene un rol específico. Juntas forman la experiencia completa que el usuario ve, siente e interactúa.

HTML

Estructura

Define qué existe en la página: títulos, párrafos, campos de formulario, botones, imágenes. Es el esqueleto. Sin HTML no hay contenido. Las etiquetas como <form>, <input> y <button> son sus bloques fundamentales para formularios.

CSS

Presentación visual

Controla cómo se ve todo: colores, tipografía, espaciado, bordes, sombras, animaciones y diseño responsivo. Sin CSS la página existe pero es gris y desordenada. Transforma el esqueleto HTML en una interfaz visual atractiva y coherente.

JavaScript

Comportamiento e interactividad

Añade lógica y dinamismo: validar que el correo tiene el formato correcto, mostrar mensajes de error en tiempo real, enviar datos al servidor sin recargar la página (fetch/AJAX), o mostrar una confirmación al usuario tras el envío exitoso.

Validación en el cliente

JavaScript revisa los datos antes de enviarlos. Ahorra tiempo al usuario y reduce peticiones innecesarias al servidor.

Envío asíncrono

Con fetch() los datos viajan al servidor en segundo plano. La página no se recarga y la experiencia es fluida.
